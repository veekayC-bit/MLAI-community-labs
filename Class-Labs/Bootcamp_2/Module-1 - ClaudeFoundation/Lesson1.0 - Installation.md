# Lesson 1.0 — Installation

## Lesson overview

In this lesson you will set up everything needed to use **Claude Code** for the rest of the course. You will:

- Install the Claude Desktop app on your Mac or Windows PC
- Enable Claude Code and sign in
- Choose a working folder for your files
- Confirm that Claude can read and write in that folder

By the end of this module, you’ll be ready to practice real product management workflows with Claude Code.

---

## Before you begin

- **Platform:** Claude Code is available on **macOS and Windows** via the Claude Desktop app.
- **Subscription:** You need a **Claude Pro** plan to use Claude Code. Confirm your account is on a Pro plan before starting.

---

## Step 1: Open the Claude Code download page

**Mac**

1. In your browser, go to: **https://claude.ai/download**
2. Click **Download for Mac** to get the macOS desktop app installer.

**Windows**

1. In your browser, go to: **https://claude.ai/download**
2. Click **Download for Windows** to get the Windows installer (`.exe`).

![Claude download page](images/0.png)

---

## Step 2: Install and sign in

**Mac**

1. Open the downloaded `.dmg` file.
2. Drag the Claude app into your **Applications** folder.
3. Open Claude from Applications or Spotlight (`Cmd + Space`, type **Claude**).
4. **Sign up or log in** with your Anthropic account (or create one if needed).

   ![Sign in](images/1.png)

   ![Sign in view](images/2.png)

**Windows**

1. Run the downloaded `.exe` installer and follow the setup prompts.
2. Open Claude from the **Start menu** or the taskbar shortcut created during installation.
3. **Sign up or log in** with your Anthropic account (or create one if needed).

---

## Step 3: Open Claude Code

After signing in (Mac or Windows):

1. In the left sidebar, **click Code** (next to Chat).

    ![Claude Code workspace](images/op.png)

2. You should see the **Claude Code workspace view**, where you can choose or create the working folder Claude will use for this course.

   ![Claude Code workspace](images/op.png)

---

## You’re ready

You’ve installed the app, signed in, and opened Claude Code. In the next lessons you’ll use this workspace to run product management workflows with Claude.
