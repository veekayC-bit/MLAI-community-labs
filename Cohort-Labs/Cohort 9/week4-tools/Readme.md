# Week 4 — AI Tools & Integrations

This week covers building  AI agents using real-world platforms and tools. Each lab is independent — you can complete them in any order, but Lab 4.2 requires the foundation setup labs to be done first.

| Lab | Title | What You Build |

## Lab 4.2 Prerequisites

Lab 4.2 connects to three external databases. Complete these setup guides and save the credentials before starting that lab:

| Setup | What You Need |
|---|---|
| [MongoDB Setup](../week%200%20%20-%20foundation/mongoDBSetup/Readme.md) | Connection string (`mongodb+srv://...`) |
| [Supabase Setup](../week%200%20%20-%20foundation/supabaseSetup/readme.md) | Project URL and Service Role Key |
| [Snowflake Setup](../week%200%20%20-%20foundation/snowflakeSetup/Readme.md) | Account ID, username, password, warehouse |

All other labs only require an OpenAI or Azure API key.
